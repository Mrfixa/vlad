## 1. Critical Blockers

**Must fix before any production deployment.**

### C1: Swish Key Rotation ⚠️ ACTION REQUIRED
- [ ] **Revoke** the live Swish merchant private key with issuer
- [ ] **Generate** new Swish certificate/key pair
- [ ] **Purge** old key from git history: `git filter-repo --path certificates/`
- [ ] **Update** `.env` with new credentials
- [ ] **Add** `certificates/live/*.key` to `.gitignore`

### C2: Demo Seeders in Production
- [ ] Gate `DefaultUsersSeeder` to non-production only
- [ ] Remove demo customer/driver accounts from production DB
- [ ] Change admin default password

### C3: CORS Configuration
- [ ] Restrict `CORS_ALLOWED_ORIGINS` to known domains
- [ ] Remove wildcard `allowed_methods` and `allowed_headers`

## 8. Deployment Checklist

### 8.1 Pre-Deployment
- [ ] All Critical blockers resolved
- [ ] All High priority items completed
- [ ] Full test suite passing
- [ ] PHPStan clean at Level 3+
- [ ] Flutter analyze clean

### 8.2 Environment Setup
- [ ] Production `.env` configured
- [ ] SSL certificates installed
- [ ] Domain pointed to server
- [ ] Database migrated
- [ ] Passport keys generated

### 8.3 Server Configuration
- [ ] PHP 8.4+ installed
- [ ] Composer dependencies installed
- [ ] Redis configured and running
- [ ] Queue worker running (Supervisor)
- [ ] Scheduler configured (crontab)

```bash
# Crontab entry
* * * * * cd /path-to-project && php artisan schedule:run >> /dev/null 2>&1
```

### 8.4 Build APKs
```bash
# User App
cd drivemond-user-app-3.1
flutter build apk --release \
  --dart-define=MAPS_API_KEY=YOUR_KEY \
  --dart-define=STRIPE_PUBLISHABLE_KEY=YOUR_KEY \
  --dart-define=BASE_URL=https://dacatlon.store

# Driver App
cd drivemond-driver-app-3.1
flutter build apk --release \
  --dart-define=MAPS_API_KEY=YOUR_KEY \
  --dart-define=BASE_URL=https://dacatlon.store
```

### 8.5 Post-Deployment
- [ ] Smoke test all API endpoints
- [ ] Verify push notifications working
- [ ] Test real-time chat
- [ ] Monitor error logs (Sentry)
- [ ] Configure production monitoring

### 8.6 Security Checklist
- [ ] `.env` not accessible via web
- [ ] `storage/` not publicly accessible
- [ ] Admin panel behind VPN or IP whitelist
- [ ] Rate limiting active
- [ ] CORS restricted
- [ ] SQL injection prevention (Eloquent ORM)
- [ ] XSS prevention (Blade escaped)
